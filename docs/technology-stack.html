KubeLab Technology Stack

Technologies powering KubeLab, a home lab for mastering DevOps, Kubernetes, Linux, and networking, supporting a GitLab CI/CD pipeline.

Orchestration

k3s

Lightweight Kubernetes for running workloads on resource-constrained nodes. Hosts Argo CD, Vault, Cert-Manager.

CI/CD

GitLab CE

Self-hosted CI/CD platform for building, testing, and deploying Java/Angular apps.

Application

Java

Backend runtime for application logic, integrated with CI/CD pipeline.

Angular

Frontend framework for building responsive web applications.

Streaming

Kafka

Distributed streaming platform for event-driven data pipelines.

Databases

MongoDB

NoSQL database for flexible, scalable data storage.

PostgreSQL

Relational database for structured data and application persistence.

Elasticsearch

Search and analytics engine for log and data indexing.

Networking

Nginx

Web server and reverse proxy for serving apps and load balancing.

Traefik

Cloud-native reverse proxy for dynamic routing in Kubernetes.

OPNsense

Open-source router/firewall for VLANs and network security.

Cilium

Networking and security for Kubernetes with eBPF.

GitOps

Argo CD

GitOps tool for continuous deployment to k3s.

Monitoring

Prometheus

Monitoring system for collecting and querying metrics.

Grafana

Visualization platform for monitoring dashboards.

Kibana

Log and data visualization tool, paired with Elasticsearch.

Security

Vault

Secrets management for secure storage of credentials.

Cert-Manager

Automated certificate management for Kubernetes TLS.

Artifact Management

Artifactory

Artifact repository for storing CI/CD build outputs.

Automation

Ansible

Automation tool for node configuration and provisioning.

Infrastructure

Terraform

Infrastructure-as-code tool for managing lab resources.

Helm

Package manager for deploying applications to k3s.

Operating System

Linux

Operating system (Ubuntu) for nodes, hosting all services.
